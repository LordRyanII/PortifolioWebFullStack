Le Renata Cafeteria

Como um bom amante de café dediquei o meu tempo em criar uma cafeteria ficticía, cujo o intuíto é simular uma cafeteria tanto no backend quanto no frontend. O Frontend é construindo com as seguintes bibliotecas e frameworks:

React: Para organização do projeto e direcionamento do frontend.

Bootstrap: Uma ótima biblioteca para construção de cards, botões... De modo profissional e responsivo.

fortawesome : Biblioteca de ícones, ótimo para adicionar detalhes minimalistas ao projeto.

Axios: Biblioteca para consumo de dados tanto no frontend, quanto backend. Nesse projeto usei para consumir dados do backend.

Já o backend é integrado com o banco de dados MongoDb, cujo sua flexibilidade ajuda e muito na questão de performance e rapidez. No backend usei as seguintes bibliotecas e frameworks:

Node.js: Plataforma de desenvolvimento web com javascript, no projeto o backend é inteiramente construído em node-js

Express: O express é um framework para node.js que permite o gerenciamento de rotas e criação de servidores usando o node. Isso tudo de uma maneira simples e rápida.

Mongoose : É uma biblioteca para gerenciar o mongoDb no node.js, aqui é usado no Crud de usuários, produtos, login, e estoque de produtos.
